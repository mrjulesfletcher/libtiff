TIFFIsBigTIFF
=============

Synopsis
--------

.. highlight:: c

::

    #include <tiffio.h>

.. c:function:: int TIFFIsBigTIFF(TIFF* tif)

Description
-----------

:c:func:`TIFFIsBigTIFF` returns a non-zero value if the TIFF file uses the
BigTIFF format.
