TIFFSwabLong
============

Synopsis
--------

.. highlight:: c

::

    #include <tiffio.h>

.. c:function:: void TIFFSwabLong(uint32_t* data)

Description
-----------

:c:func:`TIFFSwabLong` swaps the bytes in a 32-bit value pointed to by
*data*.
