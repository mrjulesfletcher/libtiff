TIFFSwabShort
=============

Synopsis
--------

.. highlight:: c

::

    #include <tiffio.h>

.. c:function:: void TIFFSwabShort(uint16_t* data)

Description
-----------

:c:func:`TIFFSwabShort` swaps the bytes in a 16-bit value pointed to by
*data*.
