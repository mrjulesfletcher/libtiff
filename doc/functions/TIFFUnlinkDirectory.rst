TIFFUnlinkDirectory
===================

Synopsis
--------

.. highlight:: c

::

    #include <tiffio.h>

.. c:function:: int TIFFUnlinkDirectory(TIFF* tif, tdir_t dirn)

Description
-----------

:c:func:`TIFFUnlinkDirectory` removes the specified directory from the
TIFF file.
